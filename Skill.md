## 使用技巧： ##
  * 指定开发者搜索
```
pub:<name>
```
> > 例:搜索Google开发的所有程序
```
pub:Google Inc.
```
  * 指定package搜索
```
pname:<package>
```
> > 例：搜索Google地图
```
pname:com.google.android.apps.maps
```
  * 直接输入Google Play应用的地址（URL）进行搜索
> > 例：Google地图
```
https://play.google.com/store/apps/details?id=com.google.android.apps.maps
```
  * 双击下载项中应用图标，打开下载保存目录（1.3.0版本支持）
